Ticket Library

A collection of common IT support issues and their possible root causes, from basic fixes to more complex solutions.

Note: To solve an issue, it's important to work through it logically rather than jumping straight to a solution or assumption. Each ticket follows a structured approach similar to real help desk environments:
Identify → Gather Information → Troubleshoot → Verify → Document.
For a more in-depth look at how I approach a real ticket from start to finish, visit my Ticketing System project.

Ticket #001

Computer Running Slow

Basic Troubleshooting

  • Check CPU, Memory & Disk usage in Task Manager
  • Disable unnecessary startup applications
  • Close unused applications and browser tabs
  • Free disk space and remove temporary files
  • Install Windows updates
  • Restart the computer
Advanced Solution

If the device is still using an HDD, recommend upgrading to an SSD to significantly improve startup and application performance.

Ticket #002

User Cannot Sign In

Basic Troubleshooting

  • Verify username and password
  • Check Caps Lock and keyboard layout
  • Confirm internet connectivity
  • Reset the user's password
  • Check if the account is locked
  • Test sign-in on another device
Advanced Solution

Verify Active Directory or Microsoft Entra ID account status, group membership, and authentication logs.

Ticket #003

No Internet Connection

Basic Troubleshooting

  • Check Wi-Fi or Ethernet connection
  • Test another website or device
  • Restart the computer and router
  • Disable/Enable the network adapter
  • Run ipconfig /release and ipconfig /renew
  • Ping the default gateway to confirm local network connectivity
  • Ping 8.8.8.8 (Google DNS) to test whether the connection reaches the internet, not just the local network
Advanced Solution

If connectivity issues continue, review DNS configuration, network adapter drivers, and router settings.

Ticket #004

Application Crashes

Basic Troubleshooting

  • Restart the application and reopen it.
  • Restart the computer.
  • Check for application updates.
  • Verify Windows is fully updated.
  • Review Task Manager for excessive CPU or memory usage.
  • Check Event Viewer for application error logs.
Advanced Solution

Reinstall the application. If crashes continue, review compatibility issues, corrupted user profiles, or escalate with application logs for further investigation.

Ticket #005

Microsoft Teams Not Working

Basic Troubleshooting

  • Confirm the internet connection is working.
  • Verify Microsoft 365 service status.
  • Sign out and back into Teams.
  • Restart the Teams application.
  • Clear the Microsoft Teams cache.
  • Verify the user's Microsoft 365 license is assigned.
Advanced Solution

Reinstall Microsoft Teams, review Microsoft Entra ID sign-in logs, verify Conditional Access policies, and confirm the user's account is not blocked or missing required licenses.

Ticket #006

Password Reset

Basic Troubleshooting

  • Verify the user's identity.
  • Confirm the username or email address.
  • Check whether the account is locked.
  • Reset the password through Microsoft Entra ID or Active Directory
  • Require a password change at next sign-in.
  • Confirm Multi-Factor Authentication is still working.
Ticket #007

Computer Won't Turn On

Basic Troubleshooting

  • Press the power button
  • Hold down the power button for a few seconds to force a restart
  • Verify the power cable is working and the device is charged
  • Confirm the power outlet works by testing another device
  • Check for power indicator lights or fan activity
  • Disconnect external devices and try powering on again
Advanced Solution

Reseat the RAM and check all internal hardware connections first. Next, test the power supply (PSU) using a PSU tester or by swapping in a known-working unit. If the issue continues, try replacing the CMOS battery. If the computer still won't turn on after these steps, the motherboard may be failing, escalate for hardware replacement.

Ticket #008

Outlook Not Sending/Receiving Email

Basic Troubleshooting

  • Verify the internet connection
  • Confirm Outlook is connected and not in Offline Mode
  • Double-check the recipient's email address for typos
  • Check if the attachment file is too large to send
  • Restart Outlook and the computer
  • Send a test email to verify the issue.
  • Check mailbox storage limits in Microsoft 365 Admin Center
Advanced Solution

Review the user's Microsoft 365 license, Exchange Online mailbox, Outlook profile, and Microsoft Entra ID sign-in logs. Escalate for further options if the issue continues, or recreate the Outlook profile if necessary.
